## เริ่มต้นใช้งาน

### การ Login
1. เปิดแอพพลิเคชัน
2. กรอก Username และ Password
3. คลิก "เข้าสู่ระบบ"

**User เริ่มต้น:**
- Admin: `admin` / `admin123`
- Cashier: `cashier` / `cashier123`

### หน้าหลัก
หลังจาก Login สำเร็จ จะเข้าสู่หน้าหลักที่มีเมนูต่างๆ:
- 📊 Dashboard - ภาพรวมระบบ
- 🛒 การขาย - เปิดหน้าจุดขาย
- 📦 สินค้า - จัดการสินค้า
- 👥 ลูกค้า - จัดการลูกค้า
- 🏪 คลังสินค้า - จัดการสต๊อก
- 📋 รายการขาย - ประวัติการขาย
- 📈 รายงาน - รายงานและกราฟ

## การขายสินค้า (POS)

### 1. เปิดหน้าจุดขาย
- คลิกเมนู "การขาย" หรือ
- กดปุ่ม **F1**

### 2. เลือกสินค้า
- คลิกที่สินค้าในตาราง หรือ
- ค้นหาสินค้าด้วยช่องค้นหา

### 3. ปรับจำนวน
- กด + เพื่อเพิ่มจำนวน
- กด - เพื่อลดจำนวน
- กด X เพื่อลบสินค้าออกจากตะกร้า

### 4. เลือกลูกค้า (Optional)
- คลิกไอคอน 👤 ที่ด้านบน
- เลือกลูกค้าจากรายการ
- หรือกด "ล้างลูกค้า" เพื่อขายแบบไม่ระบุลูกค้า

### 5. ใส่ส่วนลด (Optional)
- คลิกไอคอน 🎟️ ที่ด้านบน
- เลือกส่วนลด % หรือ ฿
- ใส่จำนวนส่วนลด
- คลิก "ตกลง"

### 6. พักบิล (Optional)
- คลิกปุ่ม "พักบิล"
- ตั้งชื่อบิล
- บิลจะถูกเก็บไว้ที่ไอคอน 📁
- สามารถเรียกกลับมาได้ทีหลัง

### 7. ชำระเงิน
- คลิกปุ่ม "ชำระเงิน"
- เลือกวิธีชำระ (เงินสด/บัตร/โอน)
- **เงินสด**: กรอกจำนวนเงินที่รับ (ระบบจะคำนวณเงินทอนให้)
- **บัตร/โอน**: ยอดชำระคือยอดรวมทั้งหมด
- คลิก "ชำระเงิน"

### 8. เสร็จสิ้น
- ระบบจะแสดง Dialog สำเร็จ
- สามารถพิมพ์ใบเสร็จได้
- สต๊อกจะถูกตัดอัตโนมัติ

## จัดการสินค้า

### เพิ่มสินค้าใหม่
1. คลิกเมนู "สินค้า" หรือกด **F2**
2. คลิกปุ่ม ➕
3. กรอกข้อมูล:
   - รหัสสินค้า *
   - ชื่อสินค้า *
   - บาร์โค้ด
   - หน่วย *
   - ราคาขาย (Level 1-5)
   - ต้นทุนมาตรฐาน
   - ✅ ควบคุมสต๊อก
   - ✅ อนุญาตติดลบ
4. คลิก "เพิ่มสินค้า"

### แก้ไขสินค้า
1. คลิกไอคอน ✏️ ที่สินค้าที่ต้องการแก้ไข
2. แก้ไขข้อมูล
3. คลิก "บันทึก"

### ลบสินค้า
1. คลิกไอคอน 🗑️ ที่สินค้าที่ต้องการลบ
2. ยืนยันการลบ

### ค้นหาสินค้า
- พิมพ์ชื่อหรือรหัสสินค้าในช่องค้นหา
- รายการจะกรองทันที

---

## จัดการลูกค้า

### เพิ่มลูกค้าใหม่
1. คลิกเมนู "ลูกค้า" หรือกด **F3**
2. คลิกปุ่ม ➕
3. กรอกข้อมูล:
   - รหัสลูกค้า *
   - ชื่อลูกค้า *
   - ที่อยู่
   - โทรศัพท์
   - อีเมล
   - เลขผู้เสียภาษี
   - วงเงินเครดิต
   - ระยะเวลาเครดิต (วัน)
   - เลขที่สมาชิก (สำหรับสะสมแต้ม)
4. คลิก "เพิ่มลูกค้า"

### การใช้งาน
- แก้ไข/ลบ เหมือนการจัดการสินค้า
- ค้นหาด้วยชื่อ, รหัส, หรือเบอร์โทร

---

## จัดการสต๊อก

### ดูสต๊อกคงเหลือ
1. คลิกเมนู "คลังสินค้า" หรือกด **F6**
2. ดูรายการสต๊อกทั้งหมด
3. 🔴 แดง = สต๊อกต่ำ
4. 🟢 เขียว = สต๊อกปกติ

### รับสินค้าเข้า (Stock In)
1. คลิกที่สินค้า
2. เลือก "รับสินค้าเข้า"
3. กรอกจำนวน
4. กรอกเลขที่เอกสารอ้างอิง (Optional)
5. กรอกหมายเหตุ (Optional)
6. คลิก "บันทึก"

### เบิกสินค้าออก (Stock Out)
1. คลิกที่สินค้า
2. เลือก "เบิกสินค้าออก"
3. กรอกจำนวน (ต้องไม่เกินสต๊อกคงเหลือ)
4. กรอกข้อมูลอ้างอิง
5. คลิก "เบิกออก"

### โอนย้ายสินค้า (Transfer)
1. คลิกที่สินค้า
2. เลือก "โอนย้ายสินค้า"
3. เลือกคลังปลายทาง
4. กรอกจำนวน
5. คลิก "โอนย้าย"

### ปรับสต๊อก (Adjust)
1. คลิกที่สินค้า
2. เลือก "ปรับสต๊อก"
3. กรอกจำนวนใหม่
4. ระบบจะแสดงผลต่าง
5. กรอกเหตุผล *
6. คลิก "บันทึก"

### ดูประวัติการเคลื่อนไหว
- คลิกไอคอน 🕐 ที่ด้านบน
- ดูรายการการเคลื่อนไหวทั้งหมด

## ดูรายงาน

### รายงานสรุป
1. คลิกเมนู "รายงาน" หรือกด **F7**
2. ดูการ์ดสรุป:
   - 💰 ยอดขายรวม
   - 🛒 จำนวนออเดอร์
   - 📊 ยอดเฉลี่ย/ออเดอร์
   - 🎫 ส่วนลดรวม
3. ดูสินค้าขายดี Top 5
4. ดูลูกค้าซื้อบ่อย Top 5

### กราฟยอดขาย
1. คลิกไอคอน 📊 ที่ด้านบน
2. ดูกราฟยอดขาย 7 วันล่าสุด
3. ดูตารางข้อมูลรายวัน

### Export รายงาน
1. คลิกไอคอน 📥 ที่ด้านบน
2. ระบบจะ Export เป็นไฟล์ CSV
3. เปิดไฟล์ด้วย Excel หรือ Google Sheets

---

## ตั้งค่าระบบ

### ข้อมูลบริษัท
1. คลิกไอคอน ⚙️ ที่ด้านบน
2. แก้ไข:
   - ชื่อบริษัท
   - เลขผู้เสียภาษี
   - ที่อยู่
   - โทรศัพท์
3. คลิก "บันทึกข้อมูลบริษัท"

### ตั้งค่า VAT
1. เปิด/ปิด VAT
2. ระบุอัตรา VAT (%)
3. คลิก "บันทึกการตั้งค่า VAT"

### ตั้งค่าสต๊อก
1. เปิด/ปิดการแจ้งเตือนสต๊อกต่ำ
2. ระบุจำนวนสต๊อกต่ำสุด
3. คลิก "บันทึกการตั้งค่าสต๊อก"

## 💡 Tips & Tricks

### Keyboard Shortcuts
- **F1** - เปิด POS (ใช้บ่อยที่สุด!)
- **F2** - จัดการสินค้า
- **F3** - จัดการลูกค้า
- **F4** - ประวัติการขาย
- **F5** - Refresh
- **F6** - คลังสินค้า
- **F7** - รายงาน
- **F10** - Dashboard
- **ESC** - ปิด/ยกเลิก

### Best Practices
1. **ตรวจสอบสต๊อกทุกวัน** - ป้องกันขาดสต๊อก
2. **บันทึกลูกค้า** - ใช้วิเคราะห์การซื้อ
3. **ตั้งค่าเตือนสต๊อก** - แจ้งเตือนเมื่อใกล้หมด
4. **Export รายงานสม่ำเสมอ** - เก็บข้อมูลสำรอง
5. **ใช้ส่วนลดอย่างระมัดระวัง** - ตรวจสอบก่อนบันทึก

### การแก้ปัญหา
**Q: Login ไม่ได้**
A: ตรวจสอบ Username/Password ให้ถูกต้อง

**Q: สินค้าไม่แสดงใน POS**
A: ตรวจสอบว่าสินค้า is_active = true

**Q: สต๊อกไม่ถูกต้อง**
A: ดูประวัติการเคลื่อนไหวเพื่อตรวจสอบ

**Q: รายงานไม่มีข้อมูล**
A: ตรวจสอบว่ามีการขายแล้ว และกด Refresh
