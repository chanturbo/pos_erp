## เริ่มต้นใช้งาน

### การ Login
1. เปิดแอพพลิเคชัน
2. กรอก Username และ Password
3. คลิก "เข้าสู่ระบบ"

**User เริ่มต้น:**
- Admin: `admin` / `admin123`
- Cashier: `cashier` / `cashier123`

### หน้าหลัก
หลังจาก Login สำเร็จ จะเข้าสู่หน้าหลักที่มีเมนูต่างๆ:
- 📊 Dashboard - ภาพรวมระบบ
- 🛒 การขาย - เปิดหน้าจุดขาย
- 📦 สินค้า - จัดการสินค้า
- 👥 ลูกค้า - จัดการลูกค้า
- 🏪 คลังสินค้า - จัดการสต๊อก
- 📋 รายการขาย - ประวัติการขาย
- 📈 รายงาน - รายงานและกราฟ

## การขายสินค้า (POS)

### 1. เปิดหน้าจุดขาย
- คลิกเมนู "การขาย" หรือ
- กดปุ่ม **F1**

### 2. เลือกสินค้า
- คลิกที่สินค้าในตาราง หรือ
- ค้นหาสินค้าด้วยช่องค้นหา

### 3. ปรับจำนวน
- กด + เพื่อเพิ่มจำนวน
- กด - เพื่อลดจำนวน
- กด X เพื่อลบสินค้าออกจากตะกร้า

### 4. เลือกลูกค้า (Optional)
- คลิกไอคอน 👤 ที่ด้านบน
- เลือกลูกค้าจากรายการ
- หรือกด "ล้างลูกค้า" เพื่อขายแบบไม่ระบุลูกค้า

### 5. ใส่ส่วนลด (Optional)
- คลิกไอคอน 🎟️ ที่ด้านบน
- เลือกส่วนลด % หรือ ฿
- ใส่จำนวนส่วนลด
- คลิก "ตกลง"

### 6. พักบิล (Optional)
- คลิกปุ่ม "พักบิล"
- ตั้งชื่อบิล
- บิลจะถูกเก็บไว้ที่ไอคอน 📁
- สามารถเรียกกลับมาได้ทีหลัง

### 7. ใช้คูปองส่วนลด (Optional)
- หน้าชำระเงินมี section "คูปองส่วนลด"
- กรอกรหัสคูปอง แล้วกด "ใช้คูปอง" หรือสแกน QR
- ใช้ได้หลายใบพร้อมกัน (ขึ้นกับเงื่อนไขโปรโมชั่น)

### 8. แลกแต้มสะสม (Optional — เฉพาะลูกค้าสมาชิก)
- หากเลือกลูกค้าที่มีเลขสมาชิก จะเห็น section "ใช้แต้มสะสม" สีส้ม
- ระบบแสดงแต้มคงเหลือ เช่น `มี 150 แต้ม (สูงสุด 120 pt)`
  - สูงสุด = min(แต้มที่มี, ยอดบิล) ไม่สามารถแลกเกินยอดได้
- กรอกจำนวนแต้มที่ต้องการใช้ แล้วกด **"ใช้แต้ม"** หรือกด **"ใช้ทั้งหมด"**
- ยอดชำระจะลดลงทันที — **1 แต้ม = ลด ฿1.00**
- กด ✕ เพื่อยกเลิกการใช้แต้มได้ก่อนชำระ

### 9. ชำระเงิน
- คลิกปุ่ม "ชำระเงิน"
- เลือกวิธีชำระ (เงินสด/บัตร/โอน)
- **เงินสด**: กรอกจำนวนเงินที่รับ (ระบบจะคำนวณเงินทอนให้)
- **บัตร/โอน**: ยอดชำระคือยอดรวมหลังหักแต้มแล้ว
- คลิก "ชำระเงิน"

### 10. เสร็จสิ้น
- ระบบแสดงใบเสร็จ thermal style ทันที
- ใบเสร็จแสดง: รายการสินค้า, ส่วนลด, คูปอง, **แต้มที่แลก**, ยอดชำระ, **แต้มที่ได้รับ**
- กดปุ่มพิมพ์ (🖨️) บนหน้า PDF เพื่อพิมพ์ได้ทันที
- สต๊อกถูกตัดอัตโนมัติ แต้มลูกค้าอัปเดตทันที

## จัดการสินค้า

### เพิ่มสินค้าใหม่
1. คลิกเมนู "สินค้า" หรือกด **F2**
2. คลิกปุ่ม ➕
3. กรอกข้อมูล:
   - รหัสสินค้า *
   - ชื่อสินค้า *
   - บาร์โค้ด
   - หน่วย *
   - ราคาขาย (Level 1-5)
   - ต้นทุนมาตรฐาน
   - ✅ ควบคุมสต๊อก
   - ✅ อนุญาตติดลบ
4. คลิก "เพิ่มสินค้า"

### แก้ไขสินค้า
1. คลิกไอคอน ✏️ ที่สินค้าที่ต้องการแก้ไข
2. แก้ไขข้อมูล
3. คลิก "บันทึก"

### ลบสินค้า
1. คลิกไอคอน 🗑️ ที่สินค้าที่ต้องการลบ
2. ยืนยันการลบ

### ค้นหาสินค้า
- พิมพ์ชื่อหรือรหัสสินค้าในช่องค้นหา
- รายการจะกรองทันที

## จัดการลูกค้า

### เพิ่มลูกค้าใหม่
1. คลิกเมนู "ลูกค้า" หรือกด **F3**
2. คลิกปุ่ม ➕
3. กรอกข้อมูล:
   - รหัสลูกค้า * (สร้างอัตโนมัติ `CUS-YYMMDD-XXX`)
   - ชื่อลูกค้า *
   - ที่อยู่
   - โทรศัพท์
   - อีเมล
   - เลขผู้เสียภาษี
   - วงเงินเครดิต / ระยะเวลาเครดิต
   - ระดับราคา (1–5)
   - **เลขที่สมาชิก** — ใส่เพื่อให้สะสม/แลกแต้มได้
4. คลิก "เพิ่มลูกค้า"

### การใช้งาน
- แก้ไข/ลบ เหมือนการจัดการสินค้า
- ค้นหาด้วยชื่อ, รหัส, หรือเบอร์โทร
- กด badge **"X แต้ม >"** ในหน้ารายละเอียดลูกค้า เพื่อดูประวัติแต้มสะสม

## ระบบแต้มสะสม (Loyalty Points)

### ภาพรวม

| รายละเอียด | ค่า |
|---|---|
| อัตราสะสม | ทุก ฿100 = 1 แต้ม (ปรับได้ใน Settings) |
| อัตราแลก | 1 แต้ม = ลดราคา ฿1.00 |
| เงื่อนไข | เฉพาะลูกค้าที่มีเลขสมาชิก (Member No) เท่านั้น |

---

### การสะสมแต้ม (EARN)

แต้มจะถูกบวกให้ลูกค้า **อัตโนมัติ** ทุกครั้งที่ชำระเงินสำเร็จ

- คำนวณจาก `ยอดชำระสุทธิ (หลังหักแต้มที่แลก) ÷ 100`
- บันทึกลง `points_transactions` ประเภท **EARN** พร้อมเลขออเดอร์อ้างอิง
- แสดงบนใบเสร็จ: "ได้รับ X แต้มสะสม" (สีทอง)

**ตัวอย่าง:** ยอดชำระ ฿350 → ได้ 3 แต้ม

---

### การแลกแต้ม (REDEEM)

1. เปิดหน้า **ชำระเงิน** และเลือกลูกค้าสมาชิก
2. เลื่อนดูหัวข้อ **"ใช้แต้มสะสม"** (กรอบสีส้ม)
3. ดูแต้มคงเหลือและสูงสุดที่แลกได้ในบิลนี้
4. กรอกจำนวนแต้ม แล้วกด **"ใช้แต้ม"** หรือกด **"ใช้ทั้งหมด"**
5. ยอดชำระลดลงทันที — กด ✕ ได้หากต้องการยกเลิก
6. กดปุ่ม **"ชำระเงิน"** — ระบบหักแต้ม + บวกแต้มใหม่พร้อมกัน

> **หมายเหตุ:** ไม่สามารถแลกแต้มเกินยอดบิล หรือเกินแต้มที่มีอยู่

---

### ดูประวัติแต้ม

1. ไปที่เมนู **ลูกค้า** → เปิดหน้ารายละเอียดลูกค้า
2. กดที่ badge **"X แต้ม >"** (สีทอง ใต้เลขสมาชิก)
3. หน้า **ประวัติแต้ม** จะแสดง:

| ส่วน | รายละเอียด |
|---|---|
| Summary bar | แต้มคงเหลือ / สะสมทั้งหมด / แลกไปทั้งหมด |
| การ์ดสีเขียว | EARN — สะสมแต้มจากการซื้อ |
| การ์ดสีส้ม | REDEEM — แลกแต้มส่วนลด |
| ข้อมูลในการ์ด | ประเภท, รายละเอียด, เลขออเดอร์อ้างอิง, วันเวลา |

---

### ใบเสร็จที่แสดงข้อมูลแต้ม

ใบเสร็จ (ทั้งหลังชำระเงินและในประวัติรายการขาย) แสดงข้อมูลแต้มครบ:

```
แลกแต้ม 50 pt        -฿50.00
────────────────────────────
ยอดชำระ               ฿250.00
────────────────────────────
ชำระด้วย    เงินสด
รับเงิน              ฿300.00
เงินทอน               ฿50.00

★ ได้รับ 2 แต้มสะสม
```

---

### การตั้งค่าแต้ม

ไปที่ **Settings → Loyalty Points**:
- เปิด/ปิดระบบแต้ม
- กำหนดอัตราสะสม: ทุกกี่บาทได้ 1 แต้ม (default: 100)
- Preset: 10 / 20 / 50 / 100 บาท/แต้ม

## จัดการสต๊อก

### ดูสต๊อกคงเหลือ
1. คลิกเมนู "คลังสินค้า" หรือกด **F6**
2. ดูรายการสต๊อกทั้งหมด
3. 🔴 แดง = สต๊อกต่ำ
4. 🟢 เขียว = สต๊อกปกติ

### รับสินค้าเข้า (Stock In)
1. คลิกที่สินค้า
2. เลือก "รับสินค้าเข้า"
3. กรอกจำนวน
4. กรอกเลขที่เอกสารอ้างอิง (Optional)
5. กรอกหมายเหตุ (Optional)
6. คลิก "บันทึก"

### เบิกสินค้าออก (Stock Out)
1. คลิกที่สินค้า
2. เลือก "เบิกสินค้าออก"
3. กรอกจำนวน (ต้องไม่เกินสต๊อกคงเหลือ)
4. กรอกข้อมูลอ้างอิง
5. คลิก "เบิกออก"

### โอนย้ายสินค้า (Transfer)
1. คลิกที่สินค้า
2. เลือก "โอนย้ายสินค้า"
3. เลือกคลังปลายทาง
4. กรอกจำนวน
5. คลิก "โอนย้าย"

### ปรับสต๊อก (Adjust)
1. คลิกที่สินค้า
2. เลือก "ปรับสต๊อก"
3. กรอกจำนวนใหม่
4. ระบบจะแสดงผลต่าง
5. กรอกเหตุผล *
6. คลิก "บันทึก"

### ดูประวัติการเคลื่อนไหว
- คลิกไอคอน 🕐 ที่ด้านบน
- ดูรายการการเคลื่อนไหวทั้งหมด

## ดูรายงาน

### รายงานสรุป
1. คลิกเมนู "รายงาน" หรือกด **F7**
2. ดูการ์ดสรุป:
   - 💰 ยอดขายรวม
   - 🛒 จำนวนออเดอร์
   - 📊 ยอดเฉลี่ย/ออเดอร์
   - 🎫 ส่วนลดรวม
3. ดูสินค้าขายดี Top 5
4. ดูลูกค้าซื้อบ่อย Top 5

### กราฟยอดขาย
1. คลิกไอคอน 📊 ที่ด้านบน
2. ดูกราฟยอดขาย 7 วันล่าสุด
3. ดูตารางข้อมูลรายวัน

### Export รายงาน
1. คลิกไอคอน 📥 ที่ด้านบน
2. ระบบจะ Export เป็นไฟล์ CSV
3. เปิดไฟล์ด้วย Excel หรือ Google Sheets

## ตั้งค่าระบบ

### ข้อมูลบริษัท
1. คลิกไอคอน ⚙️ ที่ด้านบน
2. แก้ไข:
   - ชื่อบริษัท
   - เลขผู้เสียภาษี
   - ที่อยู่
   - โทรศัพท์
3. คลิก "บันทึกข้อมูลบริษัท"

### ตั้งค่า VAT
1. เปิด/ปิด VAT
2. ระบุอัตรา VAT (%)
3. คลิก "บันทึกการตั้งค่า VAT"

### ตั้งค่าสต๊อก
1. เปิด/ปิดการแจ้งเตือนสต๊อกต่ำ
2. ระบุจำนวนสต๊อกต่ำสุด
3. คลิก "บันทึกการตั้งค่าสต๊อก"

## 💡 Tips & Tricks

### Keyboard Shortcuts
- **F1** - เปิด POS (ใช้บ่อยที่สุด!)
- **F2** - จัดการสินค้า
- **F3** - จัดการลูกค้า
- **F4** - ประวัติการขาย
- **F5** - Refresh
- **F6** - คลังสินค้า
- **F7** - รายงาน
- **F10** - Dashboard
- **ESC** - ปิด/ยกเลิก

### Best Practices
1. **ตรวจสอบสต๊อกทุกวัน** - ป้องกันขาดสต๊อก
2. **บันทึกลูกค้า** - ใช้วิเคราะห์การซื้อ
3. **ตั้งค่าเตือนสต๊อก** - แจ้งเตือนเมื่อใกล้หมด
4. **Export รายงานสม่ำเสมอ** - เก็บข้อมูลสำรอง
5. **ใช้ส่วนลดอย่างระมัดระวัง** - ตรวจสอบก่อนบันทึก

### การแก้ปัญหา
**Q: Login ไม่ได้**
A: ตรวจสอบ Username/Password ให้ถูกต้อง

**Q: สินค้าไม่แสดงใน POS**
A: ตรวจสอบว่าสินค้า is_active = true

**Q: สต๊อกไม่ถูกต้อง**
A: ดูประวัติการเคลื่อนไหวเพื่อตรวจสอบ

**Q: รายงานไม่มีข้อมูล**
A: ตรวจสอบว่ามีการขายแล้ว และกด Refresh

**Q: ไม่เห็น section "ใช้แต้มสะสม" ในหน้าชำระเงิน**
A: ต้องเลือกลูกค้าที่มีเลขสมาชิก (Member No) ก่อน — ลูกค้าทั่วไป (WALK-IN) ไม่รองรับแต้ม

**Q: แลกแต้มแล้วยอดไม่เปลี่ยน**
A: กดปุ่ม "ใช้แต้ม" หลังกรอกจำนวน — แค่กรอกตัวเลขยังไม่พอ

**Q: แต้มสะสมใน badge ไม่อัปเดต**
A: กลับไปหน้าลูกค้าแล้วเปิดหน้ารายละเอียดใหม่ หรือกด Refresh
